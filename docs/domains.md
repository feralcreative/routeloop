# RouteLoop Domain Ideas

A working checklist for picking the right domain. The app is a web tool
for riders to save **private** motorcycle routes and **share them
deliberately** with specific people by email. Good names lean on one of
three themes: the **route/loop**, the **ride/moto**, or the
**private/share** angle.

Current pick: **routeloop.app** (keep it on the shortlist — see below).

> Nothing here has been checked for availability or trademark. Tick the
> boxes as you verify each one. I can run live availability checks if you
> want.

## How to judge a candidate

Score every name you like against this before falling in love with it.

- [ ] Say it out loud — no spelling-out needed ("is that loop or loup?")
- [ ] Short enough to type one-handed at a gas stop (<= 10 chars ideal)
- [ ] No awkward double letters across a word break (e.g. `routestop`)
- [ ] Reads cleanly with the TLD as one phrase (`route.bike`)
- [ ] Not easily confused with an existing app or brand
- [ ] `.com` is gettable *or* you're fine being `.app` / `.cc` forever
- [ ] Matching social handles are free (IG, X, maybe Reddit)
- [ ] No bicycle confusion if using `.bike` (you're motorcycle, not pedal)
- [ ] Survives the "screenshot test" — looks right printed on the map UI
- [ ] You still like it after sleeping on it

## The case for keeping routeloop.app

Be honest about whether you actually need to switch.

- [ ] `.app` forces HTTPS (good for a PWA — you already ship a manifest)
- [ ] "RouteLoop" already lives in the README, primer, and brand assets
- [ ] Rebrand cost: assets, icons, repo name, copy, muscle memory
- [ ] "Loop" nicely implies a round-trip ride, not just A-to-B
- [ ] Only switch if a candidate below clears the bar *and* is available

## TLD options for any name above

Same name, different ending — sometimes the cheapest "rebrand".

- [ ] `.app` — current; HTTPS-enforced, modern, app-appropriate
- [ ] `.cc` — short, brandable, widely accepted
- [ ] `.io` — techy, pricey, common for tools
- [ ] `.co` — clean `.com` alternative
- [ ] `.bike` — thematic, but skews **bicycle**; test for confusion
- [ ] `.com` — only if the exact name is actually free (rarely is)

## Before you buy, run the full check

- [ ] Domain available at a registrar you trust (Cloudflare / Porkbun)
- [ ] No active trademark on the name (USPTO TESS quick search)
- [ ] App Store / Play Store name not already claimed
- [ ] Social handles free on the platforms you'll use
- [ ] No ugly meaning in the .com squatter's parked page
- [ ] Renewal price is sane (some `.app`/niche TLDs spike after year one)
- [ ] Decide: redirect old domain or commit fully to the new one
